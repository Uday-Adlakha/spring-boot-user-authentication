# Spring Boot Authentication Project

A secure and scalable authentication system developed using Spring Boot and Spring Security.  
This project implements core backend concepts such as user registration, login authentication, password encryption, and database integration following industry best practices.

---

## Project Overview

This project demonstrates how authentication works in a real-world backend application.  
It focuses on secure user handling, clean code structure, and Spring Security configuration.

---

## Key Features

- User Registration with validation
- User Login Authentication
- Password Encryption using BCrypt
- Spring Security-based Authentication
- Session Management
- Logout Functionality
- MySQL Database Integration
- Layered Architecture (Controller, Service, Repository)

---

## Technology Stack

- Java
- Spring Boot
- Spring Security
- Hibernate / JPA
- MySQL
- Maven

---

## Application Architecture

- Controller Layer – Handles HTTP requests
- Service Layer – Business logic
- Repository Layer – Database operations
- Model Layer – Entity classes
- Security Configuration – Authentication & authorization

## Setup and Installation

### Step 1: Clone the Repository
```bash
git clone https://github.com/your-username/spring-boot-authentication.git
```

Step 2: Configure Database
Update application.properties with your MySQL credentials

spring.datasource.url=jdbc:mysql://localhost:3306/your_database
spring.datasource.username=your_username
spring.datasource.password=your_password


Step 3: Run the Application

mvn spring-boot:run

Step 4: Access the Application

http://localhost:8080
